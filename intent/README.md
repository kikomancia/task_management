# 🎯 Intent — WHY + WHAT

## Project Vision
Build a **Kanban-style Task Management Web App** for efficient workflow visualization and task organization.

## Core Features

### 1. Task Management
- **Create tasks** with title and description
- **Move tasks** between columns (To Do → In Progress → Done)
- **Delete tasks** from any column
- **View task details** (title, description, status, timestamps)

### 2. Kanban Board
- **3-column layout**: To Do, In Progress, Done
- **Drag-and-drop** task movement between columns
- **Real-time updates** as tasks move
- **Persistent storage** in MongoDB

### 3. User Experience
- Minimal, clean interface
- Intuitive task workflow
- No authentication required (MVP)
- No task filtering/sorting (MVP)

## Success Criteria
- ✅ Tasks persist across browser refreshes
- ✅ All 3 columns visible simultaneously
- ✅ Smooth drag-and-drop experience
- ✅ Task CRUD operations work correctly

## Out of Scope (MVP)
- User authentication
- Multiple workspaces
- Task priority/labels
- Due dates
- Task assignments
- Comments/collaboration
